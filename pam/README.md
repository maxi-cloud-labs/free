`pam` is a custom pam module to deal with terminal login while leveraging the Better Auth backend which has the user credentials.

# Compilation of this folder
To compile this folder:
* OS requirement: **Linux Ubuntu** or similar
* Run `apt-get install build-essential libpam0g-dev`
* Run `make`
